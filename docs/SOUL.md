# Project Soul

## Vision
To provide the ultimate, zero-friction experience for terminal personalization, making the power of Oh My Posh accessible through a professional-grade TUI and CLI.

## Principles
1. **Safety First:** Never compromise the user's PowerShell profile; use markers, backups, and non-destructive edits.
2. **Native Speed:** Leverage Rust and Tokio for a blazing-fast, non-blocking experience.
3. **Information Density:** Design for professionals who need clear, concise, and structured data.
4. **Surgical Precision:** Allow toggling specific theme segments without destroying the overall configuration.
5. **Universal Access:** Provide both a rich TUI for exploration and a robust CLI for automation.

## Values
- **The "Unstuck" Guarantee:** Integrated timeouts and guards to ensure the tool never hangs.
- **Auditable State:** Configuration changes should be clean, well-marked, and easy to understand.
- **Developer Delight:** A tool that feels "alive," responsive, and aesthetically pleasing.
